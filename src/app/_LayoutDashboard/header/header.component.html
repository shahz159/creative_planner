Stream Planner New Features

Create Project

To create a new project in the Stream Planner, follow these steps.

  1. Navigate to the sidebar on the left-hand side of the screen.
  2. Look for the 'New Project' button within the sidebar.
  3. Click on the 'New Project' button to proceed with project creation. Once clicked, follow the on-screen prompts to complete the project setup."

Create Project

To create a project, you can choose from the following four options.

  1. Start from Scratch: Begin a new project with a blank slate, allowing for full creativity and customization from the ground up.
  2. My Drafts: Revisit and continue developing projects from your existing drafts, utilizing previously outlined ideas and concepts.
  3. Assigned/Conditional Projects: Tackle projects assigned by others or triggered under specific conditions, which come with predefined objectives and may require collaboration.
  4. Saved Templates: Quickly start projects using pre-existing templates that provide a structured format and require minimal initial setup.
